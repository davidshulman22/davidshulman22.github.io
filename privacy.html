Privacy Policy

Last updated: May 14, 2026

Scope

This policy covers the GS Bookkeeping Tool, a private internal integration developed by David A. Shulman of Ginsberg Shulman, PL (the "Firm") for the Firm's own accounting use with QuickBooks Online.

Who uses this tool

The tool is used solely by David A. Shulman and authorized agents of the Firm. It is not distributed, licensed, sold, or otherwise made available to third parties or the public.

Data accessed

The tool accesses the Firm's own QuickBooks Online company data via Intuit's authorized API, including chart of accounts, transactions, account balances, and company information. The tool runs locally on hardware owned and controlled by the Firm.

Data sharing

No data accessed through the tool is shared with, transmitted to, or sold to any third party. Data does not leave the Firm's local environment except as required to communicate with Intuit's API for the purpose of reading and writing the Firm's own accounting records.

Storage and retention

OAuth credentials are stored locally on the Firm's hardware with restricted file permissions. An audit log of API calls is retained locally for the Firm's own recordkeeping. No third-party analytics, tracking, or telemetry is used.

Security

Credentials are stored with operating-system file permissions limiting access to the user account that owns the tool. Refresh tokens rotate per Intuit's OAuth specification.
